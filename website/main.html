Game Commands

The game has several commands that you can use to interact with the game world and other players. Here's a breakdown of the most important commands:

Communication

  • msg [username] [message] - sends a message to [username]
  • yell [message] - sends a message to all players in your block

    • Movement

  • go [direction] - moves to the specified direction (N, NE, E, SE, S, SW, W, NW)
  • move [x],[y] - moves to the coordinate in the shortest distance 1 block per second, eg. travel [0,0]
  • stop - to stop the travel

    • Interaction

  • take - picks up the item
  • use [item] - uses the specified item
  • talk [message] - - initiates a conversation with an npc

    • Inventory Management

  • inv - displays your inventory
  • drop [item] - drops the specified item from your inventory

    • Special Interaction

  • search - search the current area
  • rest - rest
  • sell [item] - sell item to an npc
  • buy [item] - buy item from an npc

    • Combat

  • attack [target] - attacks the specified target
  • defend - prepares to defend against the next attack
